Pearson Correlation

An app that takes in two values (x,y) which create a data point. After entering data points, you will be able to click the calculate button, which will provide means, standard deviation, Pearson Correlation Coefficient, and a basic interpretation. Additionally, a scatter plot will be populated with your data and provide you a regression

You can enter values one by one with the add button, or you can pass in an Excel file (.xlsx) that contains the values in two columns.

Please enter a CSV file that includes the data you want to analyse in column one and two.
